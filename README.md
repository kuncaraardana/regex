
# regex

Regular expression utility functions.

## quote

```javascript
function quote(string: string): string
```

Quotes special regular expression characters: . \ + * ? [ ^ ] $ ( ) { } = ! < > | : -

## unquote

```javascript
function unquote(string: string): string
```

Unquotes special regular expression characters: . \ + * ? [ ^ ] $ ( ) { } = ! < > | : -
